# Pattern matching

`Match scrutinee [clauses]` evaluates the scrutinee, then tries each `(pattern result)` clause in order. The first pattern that matches wins, and its result is evaluated in [tail position](./tail-calls) with any bindings the pattern introduced. If nothing matches, it's an error.

Patterns are just **forms**, interpreted structurally:

| Pattern | Matches |
|---|---|
| a literal (`42`, `"hi"`, `true`, `'x'`, `{read}`) | a value equal to it |
| a bare name (`x`) | anything — and **binds** the name |
| a payload-less variant case in scope (`none`, an enum case) | that exact case, by equality |
| a call shape (`ok(n)`, `point{x: a}`) | a variant case with that label, recursing into the payload |
| a tuple `(a b)` | a tuple of the same length, element-wise |
| a list `[a b c]` | a list of the same length, element-wise |
| a record `{k: p}` | a record containing key `k` whose value matches `p` |
