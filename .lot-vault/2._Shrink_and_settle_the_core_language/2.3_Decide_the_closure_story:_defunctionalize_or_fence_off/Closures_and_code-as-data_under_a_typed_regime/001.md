# Closures and code-as-data under a typed regime

Two value families have no Canonical ABI form: functions (WIT has no
function type — TAG_FN boxes with capture lists) and the meta layer (Quote,
macros, forms). Both need a deliberate story, because "everything is ABI"
is only as true as its weakest value type.
