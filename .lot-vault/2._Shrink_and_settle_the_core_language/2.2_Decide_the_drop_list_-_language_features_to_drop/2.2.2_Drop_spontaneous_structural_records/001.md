# 2.2.2 Drop spontaneous structural records

An anonymous record literal must have a type that is declared (DefType) or
inferable from its use; a bag of ad-hoc fields flowing around untyped goes
away. Radical alternative kept on the table: row-typed records, which keep
the ergonomics at the cost of a genuinely bigger type system — recommend
nominal-plus-inference first, rows only if the ergonomic loss stings in
practice.
