# 2.2.3 Drop tag-dispatch dynamism in operations

The behaviours that read a tag at runtime and choose semantics: mixed
int-and-dec arithmetic (silent f64 promotion), match over values of
statically unknown shape, bare symbols doubling as untyped variant cases,
comparisons across types. Each becomes a compile error or an explicit
conversion (to-f64). This is the least visible drop but touches the most
code: it is the box tags' whole reason to exist.
