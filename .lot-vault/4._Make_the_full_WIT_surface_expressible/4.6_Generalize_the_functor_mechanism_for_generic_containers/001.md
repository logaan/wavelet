# 4.6 Generalize the functor mechanism for generic containers

wit.rs already implements compile-time functors (Import {elem: T} specializes
a Set interface at a concrete element type, with a shared op-table so
inference and emitted WIT cannot drift). Promote this from a special case to
the language's general answer for user-defined generic containers — wavelet's
monomorphic substitute for parametric polymorphism.
