# Variants as numeric discriminant plus canonical payload

Replace TAG_VAR boxes, which carry the *case name as a runtime string box*,
with the ABI form: numeric discriminant + payload at canonical offset.
Match compiles to an integer branch table instead of string comparison.
option/result/enum/flags follow the same collapse (flags become a bare i32
bitset, which lift_flags/lower already treat as the boundary form).
