# Records and tuples in canonical memory layout

Replace TAG_REC boxes (n-fields + (key-string-box, value-box) pairs — field
names as runtime strings!) and TAG_TUP boxes with canonical struct layout:
fields at despec-computed offsets and alignments, field access compiled to a
constant-offset load. Field names exist only at compile time.
