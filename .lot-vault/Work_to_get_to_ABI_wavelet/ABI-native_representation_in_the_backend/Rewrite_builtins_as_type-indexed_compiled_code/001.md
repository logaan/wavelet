# Rewrite builtins as type-indexed compiled code

The runtime support code in emit.rs (eq, to-string, print, compare) is a
family of tag-switch loops over boxes. Replace each with per-type compiled
paths selected at compile time by the operand's static type — the codegen
counterpart of the typed builtin signatures in the polymorphism category.
