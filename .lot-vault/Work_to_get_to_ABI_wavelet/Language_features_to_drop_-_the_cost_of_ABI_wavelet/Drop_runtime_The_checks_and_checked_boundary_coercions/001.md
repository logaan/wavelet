# Drop runtime The checks and checked boundary coercions

The becomes pure static ascription (it already is for literals). The
compiler-inserted checked coercions at boundaries — and the safely wrapper
whose job is catching their traps — become dead weight for wavelet-side
values, which are correct by construction. safely survives only where
genuinely dynamic data enters: parsing input with read, foreign components
lying about their types.
