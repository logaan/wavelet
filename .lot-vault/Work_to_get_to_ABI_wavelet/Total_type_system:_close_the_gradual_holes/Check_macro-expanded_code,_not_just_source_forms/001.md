# Check macro-expanded code, not just source forms

User macro calls are opaque to the checker (Unknown, arguments deliberately
unchecked because they are forms). Run the checker again after expand.rs
reaches fixpoint so that everything macros *generate* is typed. Macro
templates themselves are data and stay uncheckable; their output is not.
