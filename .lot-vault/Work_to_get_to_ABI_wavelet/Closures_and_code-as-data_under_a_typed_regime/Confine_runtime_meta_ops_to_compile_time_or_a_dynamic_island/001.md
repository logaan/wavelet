# Confine runtime meta ops to compile time or a dynamic island

read/apply/expand at runtime produce values whose types cannot be known
statically. Macros already run at compile time (expand.rs fixpoint), so the
expander keeps full dynamism — it can stay on the interpreter. For genuinely
runtime evaluation, either drop it (see the drop-list category) or confine
it to a dedicated "dynamic island" component that keeps the boxed
interpreter and talks to typed components only through WIT boundaries —
which the boundary checks already make safe.
