# Sweep the locked example suite

Many documented examples deliberately exercise runtime type errors (non-bool
If conditions, The failures at runtime, heterogeneous-list tricks) and the
checker currently contorts itself to keep them green. Under strict mode they
become compile errors: rewrite each affected example in
docs/scripts/gen-examples.mjs to show the new compile-time diagnostic, then
./scripts/regen-examples.sh to re-lock examples.json and the wasm artifact.
