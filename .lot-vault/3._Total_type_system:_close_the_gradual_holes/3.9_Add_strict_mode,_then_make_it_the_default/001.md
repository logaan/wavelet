# 3.9 Add strict mode, then make it the default

The ratchet: a compiler flag under which Type::Unknown is an error rather
than a gradual top. Ship it opt-in first (the example suite will light up),
burn down the failures category by category, then flip the default. This is
the single gate that turns "gradual" into "total".
