# 3.1 Type qualified imports from their WIT signatures

Node::Qsym (alias/fn) types as Unknown, but every import has a declared WIT
signature the toolchain already parses (witdep.rs). Feed those signatures into
the checker so cross-component calls are as statically typed as local ones —
this is the Component Model handing us types for free.
