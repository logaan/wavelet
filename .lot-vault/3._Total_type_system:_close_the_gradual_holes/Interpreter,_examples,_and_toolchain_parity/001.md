# Interpreter, examples, and toolchain parity

interp.rs is the semantics oracle and emit.rs is validated against it, so a
change of this size is really a change to the *language*, made in the
interpreter first. Meanwhile docs/examples.json locks current behaviour into
cargo test, and several downstream surfaces (docs prose, LSP, grammars) track
the language by hand.
