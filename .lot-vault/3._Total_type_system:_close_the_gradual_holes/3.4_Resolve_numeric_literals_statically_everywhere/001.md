# 3.4 Resolve numeric literals statically everywhere

IntLit/FloatLit currently default to s64/f64 when unconstrained and are
range-checked partly at compile time (The, int_in_range) and partly at
runtime. Make every literal land on one concrete width at compile time so no
runtime range check or int-vs-dec dispatch remains.
