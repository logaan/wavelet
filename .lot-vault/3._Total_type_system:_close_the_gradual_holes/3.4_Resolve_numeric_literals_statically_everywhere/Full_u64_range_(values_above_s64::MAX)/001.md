# Full u64 range (values above s64::MAX)

Wavelet ints are i64, so u64 values above 9223372036854775807 are
unrepresentable as literals and wrap targets (u8-u32 wrap via If eq(MAX) 0;
u64 cannot). Seed B of the rust caller uses u64::MAX and will exercise this
against a future wavelet callee. Needs a representation/design decision —
flagging per the stage-3 ground rules before starting.
