# 3.7 Type the meta layer as tree-to-tree

Forms already have a WIT interchange type (the arena/tree encoding — the
design doc: "the moment a form crosses a boundary, it is a tree"). Make that
the *static* type of the meta layer too: Quote/Quasi produce tree, DefMacro
templates check as tree -> tree, form accessors (form-kind, rec-key, ...)
are ordinary typed functions over tree. Homoiconicity survives intact — code
as data becomes code as *typed* data.
