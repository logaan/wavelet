# 7.4 Support a single import that is both a runtime dependency and a macro library

A single import that is both a runtime dependency and a macro library is an
unsupported edge case (it would need two surfaces — a runtime import in the
world plus a compile-time instantiation). wit.rs:803-804, build.rs:81.
