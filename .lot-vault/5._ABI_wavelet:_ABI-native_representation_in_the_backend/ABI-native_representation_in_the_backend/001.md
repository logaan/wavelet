# ABI-native representation in the backend

Replace emit.rs's uniform tagged boxes (i32 pointer -> [tag, payload] in
linear memory, TAG_BOOL..TAG_CHAR) with type-directed representation: the
Canonical ABI layout *is* the internal layout. Values with flat ABI forms
live in wasm locals; aggregates live in linear memory at canonical
offsets/alignments. The existing lift/lower code paths become the spec to
delete against.
