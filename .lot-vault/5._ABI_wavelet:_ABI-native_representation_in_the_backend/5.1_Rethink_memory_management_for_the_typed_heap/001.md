# 5.1 Rethink memory management for the typed heap

The bump allocator leans on uniform boxes (every heap object starts with a
tag). Canonical layout has no headers, so GC/refcounting needs a different
story: per-type layout maps driven by static types, arena-per-call
(components are short-lived instances), or the design.md plan of wasm-GC
structs for the non-ABI residue (closures). Decide before the layout work
lands, since it constrains where headers may exist.
