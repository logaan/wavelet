# 5.6.1 Statically resolve arithmetic to per-width wasm ops

emit.rs currently compiles add to a runtime tag check (both ints -> checked
i64, any dec -> f64). With concrete operand types at every call site, emit
i32.add/i64.add/f32.add/f64.add (plus width-specific overflow checks)
directly. This is where unboxing pays: arithmetic on locals with zero memory
traffic.
