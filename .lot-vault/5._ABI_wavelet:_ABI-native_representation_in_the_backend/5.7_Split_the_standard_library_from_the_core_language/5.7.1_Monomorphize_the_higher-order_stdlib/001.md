# 5.7.1 Monomorphize the higher-order stdlib

map/filter/fold/zip are polymorphic in both element type and function
argument. Specialize them per concrete element type at each call site
(compile-time template expansion — they can become macros that expand to
typed tail-recursive loops), so a list<u8> map is a tight typed loop, not a
generic walk over boxes.
